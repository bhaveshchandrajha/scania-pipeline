1. Inputs & Global Context

Point to your AST directory (with *-ast.json files) and the RPG root directory (e.g. extracted PoC_HS1210). Then click Build Global Context. Neo4j Cypher export is written to global_context/neo4j_export.cypher.

4. Build Application

        
Compile and fix build errors. Runs Maven with an LLM-based autofix loop (up to 4 passes) to resolve compilation errors. Run after migrating features in tab 2. When build succeeds, use Push to Repository to push the generated code to scania-java-v2 — GitHub Actions will run tests on the pushed code.

Human-in-the-Loop (HITL)

          
When a build fails, the LLM proposes fixes. This panel shows the exact changes (before/after diff) for each file. Review the proposed fixes, then click Apply All to accept. You can apply again if the build still fails after the first or second attempt. Click Reject to discard.
