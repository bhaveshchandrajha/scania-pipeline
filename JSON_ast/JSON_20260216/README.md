# New AST directory (e.g. 20260216)

Place your **RPG program AST** files here (e.g. `HS1210-ast.json`, `HS1212-ast.json`).

- Files must match the pattern `*-ast.json`.
- After adding files, click **Discover Directories** in the Pure Java Pipeline UI; this directory will appear in the **AST Directory** dropdown.
- For **display file (DDS) ASTs** (e.g. `HS1210D-ast.json`), use the delivery folder (e.g. `HS1210D_20260216`) at project root; the UI also discovers those and lists them with a "DDS" label.
